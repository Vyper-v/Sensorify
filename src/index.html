Bienvenido a Sensorify

Esta aplicación permite realizar una serie de pruebas sobre el sensor de aceleración.

Para empezar, selecciona el sensor en el menu de navegación y pulsa el botón "start".

Una vez que el sensor esté activo, se mostrará una gráfica con los datos obtenidos.

Para finalizar, pulsa el botón "stop".

Si quieres descargarlo en formato CSV, pulsa el botón "save".

Si quieres volver a empezar, pulsa el botón "reset".

Si quieres ver los resultados de las pruebas, pulsa el botón "Results".
